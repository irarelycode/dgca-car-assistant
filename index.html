Deploy on GitHub – 3 steps

  1. Publish your Sheet: File → Share → Publish to web → choose Sheet1 → CSV. Copy the link.
  2. Edit config: Open config.js and paste your CSV URL as SHEET_CSV_URL.
  3. Push to GitHub: Create repo, upload these files, enable Settings → Pages → Deploy from main branch. Done.

Your site will fetch the sheet live on every load. Updates in Google Sheets appear instantly (no rebuild).
